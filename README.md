Descripción
-----------
ToDoAppSwiftUI es una aplicación desarrollada con SwiftUI que combina un gestor de tareas (TODO) con una pestaña financiera para registrar ingresos y egresos. Permite crear, editar y eliminar tareas y llevar un control sencillo de movimientos económicos basados en un monto base (por ejemplo, un salario).

Características
---------------
- Crear, editar y eliminar tareas.
- Marcar tareas como completadas.
- Pestaña de finanzas para registrar ingresos y egresos.
- Cálculo y visualización de saldo según una base (salario).
- Interfaz construida con SwiftUI (diseño adaptativo).

Requisitos
----------
- Xcode 12 o superior.
- iOS 14 o superior (o el SDK mínimo que soporte SwiftUI en tu proyecto).
- Swift 5.x.

Instalación
----------
1. Clona el repositorio:
   git clone https://github.com/Brandonv02/ToDoAppSwiftUI.git

2. Abre el proyecto en Xcode:
   - Haz doble clic en ToDoAppSwiftUI.xcodeproj o abre la carpeta en Xcode.

3. Selecciona un simulador o un dispositivo físico y ejecuta la aplicación (Cmd+R).

Uso
---
- Navega a la pestaña "Tareas" para:
  - Añadir nuevas tareas.
  - Editar título, descripción y estado.
  - Eliminar tareas que ya no necesites.
- Navega a la pestaña "Finanzas" para:
  - Registrar ingresos y egresos.
  - Definir una base (salario) y ver el saldo resultante.

Estructura (alta nivel)
-----------------------
- Interfaz creada con SwiftUI.
- Gestión de datos local (UserDefaults/CoreData/otra según implementación — revisar el código para detalles).
- Patrón sencillo de separación entre vista y lógica (por ejemplo MVVM si aplica).
